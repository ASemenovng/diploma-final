# Сборка отчета в Overleaf

Каталог содержит самодостаточный файл `main.tex`. Для сборки:

1. Создайте пустой проект в Overleaf.
2. Загрузите только файл `main.tex`.
3. В меню проекта выберите компилятор `pdfLaTeX`.
4. Если в проекте уже есть файл `output.pdf`, удалите или переименуйте его.
5. Нажмите `Recompile`.

Внешние изображения, библиография и дополнительные `.tex`-файлы не нужны.
Не используйте старый шаблон `/Users/a.i.semenov/Downloads/main.tex`: он
относится к предыдущему документу и содержит ссылку на отсутствующий рисунок
`example-image`.
