# План рефакторинга читаемого чистового кода

1. Перенести исследовательские MNT4-варианты из `src/` в `research_variants/`, обновить импорты gas-тестов.
2. Перенести исследовательские lollipop305-варианты из `src/` в `research_variants/`, обновить импорты gas-тестов.
3. Провести dependency-аудит production `src/`, удалить только файлы без ссылок и без роли в воспроизводимости результатов.
4. Добавить русские файловые, функциональные и константные комментарии в production Solidity-модули.
5. Добавить русские пояснения в исследовательские варианты и тесты там, где без них непонятна цель сравнения.
6. Обновить README и обзор чистовой директории.
7. Выполнить общий прогон `./scripts/run_all.sh`, сравнить ключевые gas-метрики с зафиксированным baseline и удалить build-артефакты.
