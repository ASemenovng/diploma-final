# Задание для deep agent: выделить ключевые теоретические результаты по lollipop-305 для текста диплома

## 1. Контекст

В проекте исследуется lollipop-305-158 из ePrint 2024/1627 как исследовательская альтернатива MNT4-753. Уже есть подробный документ `lollipop305_g2_twist_spec.pdf`, но для диплома он слишком длинный. Нужно выделить из него только основные теоретические результаты, пригодные для включения в итоговую работу.

Цель итогового текста: кратко и академично показать, что lollipop-направление было исследовано не только экспериментально, но и математически: зафиксированы параметры, степень вложения, twist-модель, определение `G2`, невырожденность и связь с будущей реализацией verifier.

## 2. Что нужно подготовить

Нужно создать отдельный markdown-документ на русском языке объемом примерно 3--5 страниц:

```text
LOLLIPOP305_THEORETICAL_RESULTS_RU.md
```

Текст должен быть не полным повтором исследования, а сжатым набором результатов для диплома.

## 3. Обязательная структура документа

### 3.1. Параметры lollipop-305-158

Кратко указать:

```text
x,
p = x^2 - x + 1,
q = x^2 + 1,
r,
t = x,
D,
k = 4.
```

Пояснить, почему кандидат исследовательский, а не production-secure.

### 3.2. Лемма о степени вложения

Сформулировать и доказать кратко:

```text
ord_r(p) = 4.
```

Показать:

```text
r | p^2 + 1,
r ∤ p - 1,
r ∤ p + 1,
r ∤ p^2 - 1.
```

### 3.3. Квадратичный твист

Сформулировать результат:

```text
Fp2 = Fp[u]/(u^2+1),
Fp4 = Fp2[v]/(v^2-u),
E: y^2 = x^3 + ax + b,
E': Y^2 = X^3 - aX + b u.
```

Дать отображение:

```text
psi(X,Y) = (uX, u v Y).
```

Пояснить смысл: `G2` можно хранить над `Fp2`, а вычисления проводить в `Fp4` после untwist.

### 3.4. Определение G1 и G2

Сформулировать:

```text
G1 = E(Fp)[r],
G2 = E'(Fp2)[r],
psi(G2) subset E(Fp4)[r].
```

Указать порядок twist-группы:

```text
#E'(Fp2) = x^4 - 2x^3 + 2x^2,
r | #E'(Fp2).
```

### 3.5. Невырожденность

Кратко сформулировать основной результат:

```text
If P in G1\{O}, Q in psi(G2)\{O}, then e(P,Q) != 1
```

при корректном выборе генераторов.

Доказательство дать через разные собственные подпространства Фробениуса:

```text
pi(P)=P,
pi(Q)=[p]Q,
1 != p mod r.
```

### 3.6. Ate loop и line functions

Кратко указать:

```text
T = t - 1 = x - 1.
```

Пояснить, что Miller loop строится на точке `Q` из второй группы, а line functions вычисляются в точке `P`.

### 3.7. Теоретические оптимизации

Кратко описать:

1. twist representation;
2. denominator elimination;
3. prepared line coefficients;
4. sparse line multiplication;
5. final exponent decomposition.

Важно: sparse multiplication нужно описать как известную оптимизацию pairing-реализаций, а вклад работы -- как адаптацию и измерение в EVM/Solidity/Yul для конкретных башен полей.

### 3.8. Связь с результатами проекта

Кратко связать теорию с уже реализованным кодом:

- 2-limb arithmetic дает выигрыш на `Fp/Fp2/Fp4`;
- полный verifier пока зависит от завершения `G2/twist` test vectors;
- lollipop-305 не является production-secure, но полезен как исследовательский sanity-check идеи меньших полей.
