# mnt_cycle_full

Финальный модуль из дипломной поставки. Сборочные артефакты `out/`, `cache/`, `target/` удалены намеренно: они воспроизводятся тестами.

Запуск из корня поставки:

```bash
./scripts/run_mnt_cycle_full.sh
```

Если скрипта с таким именем нет, см. список команд в корневом `README.md`.
