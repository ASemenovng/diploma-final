[`docs/MNT4_BLOCK_COMPRESSED_ORDINARY_FRI_FORMAL_SPEC_RU.md`](docs/MNT4_BLOCK_COMPRESSED_ORDINARY_FRI_FORMAL_SPEC_RU.md).

gas-эксперимента.
