# Наивная модель стоимости Tate-сопряжения MNT4-753

Этот каталог содержит не полный исполняемый контракт сопряжения, а намеренно
медленную модель стоимости исходного алгоритма. Ее задача: измерить цену
математически корректных микроблоков до применения оптимизаций.

## Что исполняется

Контракт `src/MNT4TatePairingNaive.sol` измеряет:

1. школьное умножение в `Fq`, `Fq2` и `Fq4`;
2. общий шаг цикла Миллера
   `f <- f^2 * line`;
3. 16-битный фрагмент обычного бинарного возведения в степень в `Fq4`.

В этой модели намеренно отсутствуют:

- Yul hot path;
- сокращенный ate-loop;
- подготовленные fixed-Q линии;
- разреженные умножения;
- Frobenius-разложение финальной экспоненты.

## Почему полный наивный вызов не запускается

Полный Tate-контур состоит из большого числа одинаковых шагов Миллера и
длинного бинарного возведения в степень. Наивное исполнение превышает
практический диапазон одного EVM-вызова. Поэтому итоговая стоимость вычисляется
строгой экстраполяцией измеренных блоков:

```text
gas_naive =
    N_miller * gas(generic_miller_step)
  + N_square * gas(Fq4.square)
  + N_mul * gas(Fq4.mul)
  + gas(curve_and_line_overhead).
```

Экстраполяция является нижней оценкой: она не включает все накладные расходы
построения линий и арифметики twist-точек.

Измеренные блоки дают:

```text
Tate Miller accumulator       >=   912,988,940 gas
Generic binary final exponent >= 1,635,405,741 gas
Итого                         >= 2,548,394,681 gas
```

Полный исполняемый MNT4 reference-контур с проверкой против arkworks находится
в `../../implementations/full_onchain_mnt4`.
